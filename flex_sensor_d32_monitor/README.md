# ESP32 Flex Sensor Monitor

Минимальный тестовый проект для `ESP32`, который читает flex-сенсор с `GPIO32`
и печатает в монитор порта:

- `raw` значение ADC
- напряжение на входе
- вычисленное сопротивление flex-сенсора

Схема соответствует SparkFun:

- flex-сенсор подключен к `3.3V`
- точка между flex-сенсором и резистором идет на `GPIO32`
- резистор `47k` подключен между этой точкой и `GND`

Запуск:

1. Откройте эту папку как отдельный PlatformIO-проект.
2. Прошейте плату.
3. Откройте Serial Monitor на `115200`.
